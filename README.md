# Taschenrechner

Ein moderner Web-Taschenrechner mit dunklem Design.

## Funktionen

- Grundrechenarten (+, −, ×, ÷)
- Prozent, Vorzeichenwechsel, AC
- Deutsche Zahlenformatierung
- Tastaturunterstützung

## Lokal starten

```bash
python3 -m http.server 8765
```

Dann im Browser: http://localhost:8765/
